# Contributing to NurtureFolio

Thanks for your interest in contributing! 🎉

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/FidelityNurture.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Run tests: `python tests/test_core.py`
6. Commit: `git commit -m "Add your feature"`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request

## What to Contribute

- 🐛 **Bug fixes** — Found something broken? Fix it!
- ✨ **New features** — Real market data integration, new optimization strategies
- 📖 **Documentation** — Improve README, add examples, write tutorials
- 🧪 **Tests** — More test coverage is always welcome
- 🎨 **UI improvements** — Better terminal charts, web dashboard

## Code Style

- Python 3.10+
- Use type hints
- Keep functions focused and documented
- Follow existing code patterns

## Questions?

Open an issue or reach out on GitHub!
